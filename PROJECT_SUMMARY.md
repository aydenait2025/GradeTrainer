## 🛠️ **立即开始使用**

### 快速部署 (5分钟启动)

```bash
# 1. 下载项目
cd ai-tuning-system

# 2. 配置环境
cp .env.example .env
# 编辑 .env 文件设置数据库密码等

# 3. 一键部署
./deploy.sh docker

# 4. 访问系统
# 前端: http://localhost:3000
# 后端: http://localhost:8000/docs
```

### 开发环境 (本地开发)

```bash
# 后端开发
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

# 前端开发
cd frontend
./start.sh dev
```
